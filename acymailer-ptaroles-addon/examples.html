AcyMailing PTA Roles Add-on - Email Examples

These examples show how to use the dynamic text options in your AcyMailing email templates.

Example 1: Personal Welcome Email

This email template welcomes users and shows their personal PTA involvement:

Subject: Welcome to PTA Communications, {user:name}! Dear {user:name}, Welcome to our PTA email updates! Your Current PTA Roles: {ptaroles:user_roles_detailed} You are currently involved in {ptaroles:user_role_count} role(s). Thank you for your dedication to our school community! We currently have {ptaroles:open_roles_count} volunteer opportunities available. If you're interested in taking on additional roles, please let us know. Best regards, PTA Communications Team

Example 2: Monthly Directory Newsletter

A comprehensive newsletter including the full PTA directory:

Subject: PTA Directory - {date:format:F Y}

Monthly PTA Directory Update

Quick Stats: • Total PTA Roles: {ptaroles:total_roles} • Filled Positions: {ptaroles:filled_roles} • Active Volunteers: {ptaroles:volunteer_count} • Open Positions: {ptaroles:open_roles_count}

Executive Board

{ptaroles:executive_directory}

Committee Chairs

{ptaroles:committee_chairs}

Complete Directory

{ptaroles:full_directory_detailed} Thank you to all our volunteers!

Example 3: Volunteer Recruitment Email

Targeted email to recruit volunteers for open positions:

Subject: Help Needed - {ptaroles:open_roles_count} PTA Positions Available Dear PTA Family, We need your help! Our PTA currently has several open positions that are crucial to our school's success. Open Positions Needing Volunteers: {ptaroles:open_roles} Why Volunteer? • Make a direct impact on your child's education • Connect with other parents • Contribute to school community • Gain valuable experience {if:user_roles_count > 0} Your Current Involvement: {ptaroles:user_roles} Thank you for your existing contribution! Even if you're already involved, consider if you can take on one more role. {endif} To volunteer or learn more about these positions, please reply to this email or contact us at pta@school.edu. Together we make a difference! PTA Volunteer Coordinator

Example 4: Executive Board Update

Update email specifically for executive board members:

Subject: Executive Board Update - {date:format:M j, Y} Dear Executive Board Members, Here's our current status update: Board Composition: {ptaroles:executive_directory} Current Priorities: We have {ptaroles:open_roles_count} positions that need immediate attention: {ptaroles:open_roles} Volunteer Engagement: • Total active volunteers: {ptaroles:volunteer_count} • Filled positions: {ptaroles:filled_roles} out of {ptaroles:total_roles} • Fill rate: {math:percentage:{ptaroles:filled_roles}:{ptaroles:total_roles}}% {if:user_role_count > 1} Thank you {user:name} for managing {ptaroles:user_role_count} roles: {ptaroles:user_roles} {endif} Next meeting: [DATE] at [TIME] Agenda to follow. Best, PTA President

Example 5: Simple Directory Card

Clean, simple directory format suitable for printing or quick reference:

Subject: PTA Quick Reference Directory

PTA Directory - Quick Reference

{ptaroles:full_directory}
Total Volunteers: {ptaroles:volunteer_count} | Filled Roles: {ptaroles:filled_roles}/{ptaroles:total_roles} | Open Positions: {ptaroles:open_roles_count} For questions or updates, contact pta@school.edu

Available Shortcodes Reference

Shortcode Description Output Type
{ptaroles:user_roles} User's assigned roles (simple list) Text
{ptaroles:user_roles_detailed} User's roles with descriptions HTML
{ptaroles:user_role_count} Number of user's roles Number
{ptaroles:open_roles} List of open positions HTML
{ptaroles:open_roles_count} Number of open positions Number
{ptaroles:executive_directory} Executive board directory HTML
{ptaroles:committee_chairs} Committee chairs directory HTML
{ptaroles:full_directory} Complete directory (simple) HTML
{ptaroles:full_directory_detailed} Complete directory with descriptions HTML
{ptaroles:total_roles} Total number of roles Number
{ptaroles:filled_roles} Number of filled roles Number
{ptaroles:volunteer_count} Total number of volunteers Number

Tips for Using the Add-on

  • Test your emails: Always send test emails to verify the dynamic content appears correctly
  • User-specific content: Only works for recipients who exist as WordPress users
  • Fallback content: User-specific shortcodes show "No assigned roles" if the user has no PTA roles
  • Performance: Organization-wide content (directories, stats) are generated once per campaign
  • Styling: The HTML output includes CSS classes that you can style in your email templates
  • Mobile-friendly: The generated content is responsive and works well on mobile devices
